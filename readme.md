# express实战
    基于express4.x封装而成的nodejs api框架，可用于大型项目的api开发。

# 目录结构
    .
    ├── app
    │   ├── controller      控制器
    │   ├── library         公共函数
    │   ├── middleware      中间件
    │   └── routes          路由
    ├── config              配置文件
    │   ├── index.js
    │   └── redis.js
    ├── LICENSE
    ├── logs                日志目录
    │   └── 2019-7-8.log
    ├── main.js             nodejs入口文件
    ├── node_modules        npm依赖包
    ├── package.json

# 开始运行
    npm install
    node main.js
    访问localhost:3000
